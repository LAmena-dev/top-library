# TOP-library
A practice project of a library created with HTML, CSS and JavaScript.

## Technology used
* HTML
* CSS (Grid, Flexbox)
* JavaScript (Constructors, Object Methods, DOM Manipulation)

## Features
* Comes with 3 sample books
* Ability to add books to the library
* Denote if a book is read
* Remove a book from the library
